How to disable inbound remote printing

This guide shows how to mitigate the PrintNightmare vulnerability using Microsoft’s Group Policy. PrintNightmare is a security vulnerability in the Microsoft Windows operating system. PrintNightmare allows an attacker to use Windows Print Spooler service to remotely control a computer.

To reduce the risk of an attacker using this vulnerability,
you can disable inbound remote printing using Microsoft’s Group Policy feature.

Disable the print spooler

  1. On the domain controller, select Start > Tools > Group Policy Management.

  2. In Group Policy editor, go to Computer Configuration > Administrative Templates > Printers.

  3. Scroll down the Setting and double-click Allow Print Spooler to accept client connections.

  4. Select Disabled.

  5. Click on Apply and then OK.

Restart the print spooler

You must restart the print spooler to execute the change.

  1. Go to Windows Start menu and search “Services".
  2. Double-click “Printer Spooler” in the Services list.
  3. Click Restart.
  4. Click OK.
